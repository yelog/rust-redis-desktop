## 📝 使用说明

### 编辑连接
1. 鼠标悬停在连接上
2. 点击 "✏️ Edit" 按钮
3. 修改配置
4. 点击 "💾 Update" 保存

### 删除连接
1. 鼠标悬停在连接上
2. 点击 "🗑️ Delete" 按钮
3. 连接将被删除
